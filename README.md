# Qtchat

Qtchat 是一个基于 Qt/C++ 的桌面即时通讯系统项目，目标是实现类似微信/QQ的基础聊天功能，并逐步扩展 AI 学习助手能力。

## 项目目标

- 多用户注册与登录
- 好友列表与会话列表
- 私聊消息收发
- 聊天历史记录
- AI 学习助手作为特殊联系人
- 后续扩展群聊、图片/文件发送、断线重连、本地缓存等功能

## 技术栈

- 客户端：Qt 6 / C++
- 构建系统：CMake
- 编译器：MinGW 64-bit
- 版本管理：Git / GitHub
- 开发工具：Qt Creator / VS Code / Codex
